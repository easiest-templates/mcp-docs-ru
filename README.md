# mcp-docs-ru

Шаблон лендинга для dev-tool / JS-библиотеки на русском языке. Основан на дизайне mcp.js.

![Screenshot](screenshot.png)

## Особенности

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Кириллические шрифты (Roboto + Roboto Mono)
- Адаптивный дизайн
- Темная тема

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
